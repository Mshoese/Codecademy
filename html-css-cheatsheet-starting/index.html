Common CSS Properties

References

Property Description Syntax
background-color Specifies the background color of an element background-color: color|transparent|initial|inherit;
background-image Specifies one or more background images for an element background-image: url|none|initial|inherit;
color Sets the color of text color: color|initial|inherit;

Example CSS Values

Property Example Values Usage
background-color CSS Colors body {background-color: coral;}
background-image linear-gradient(red, yellow) body { background-image: url("img_tree.gif"), url("paper.gif"); }
color CSS Colors p{color: purple;}
